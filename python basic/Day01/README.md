# 파이썬?
프로그램을 입문하기 위한 학습용 프로그램으로 시작

## 파이썬을 선택한 이유?
1. 오픈소스, 많은 라이브러리 존재
2. 배우기 쉬움

### 다양한 분야에서 활용
1. 웹서버(웹사이트 개발)
2. 크롤링
3. 업무 자동화
4. 데이터 분석
5. 인공지능
6. 게임 제작

### 단점
* 퍼포먼스 안좋음(속도 느림) → 교육용으로 개발

### 자바(JAVA), C 등
프로그래밍 언어 → 컴파일러 → 기계어 → 동작

### 파이썬
파이썬 → 인터프리터 → 기계어 → 동작
- 오류가 발생하면 그 다음부터 실행이 안됨
